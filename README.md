# Interview Knowledge Base

This repository contains structured interview preparation materials in Markdown format.

The content is organized by programming languages, technologies, and general software engineering topics. Each main language or technology should contain separate files for the following levels:

- Junior
- JuniorPlus
- Middle
- Senior

Each file uses a question-answer format:

```md
## Question 001

### Question
What is the concept?

### Answer
A practical explanation with examples, trade-offs, and common mistakes.
```

- self-assessment;
- technical screening;
- flashcards;
- future conversion into a website or searchable knowledge base;
- possible RAG usage with local or cloud LLM tools.
